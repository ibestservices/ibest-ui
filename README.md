# IBest-ui

## 简介

IBest-ui是一个简单易用的鸿蒙开源UI库。

## 下载安装

`ohpm install @ibestservices/ibest-ui`

OpenHarmony ohpm 环境配置等更多内容，请参考[如何安装 OpenHarmony ohpm 包](https://gitee.com/openharmony-tpc/docs/blob/master/OpenHarmony_har_usage.md)
